В этом коде мы используем команду powershell с опцией -Command для выполнения команд в PowerShell. Первая команда Get-NetAdapter получает информацию о 
сетевом адаптере. Затем мы используем команду Set-NetAdapter для изменения MAC-адреса выбранного сетевого интерфейса на новый MAC-адрес, указанный через 
флаг -newMac. Наконец, команда Enable-NetAdapter включает сетевой интерфейс после изменения MAC-адреса.

Вы можете запустить программу с помощью команды go run main.go -iface eth0 -newMac 00-11-22-33-44-55, заменив "eth0" и "00-11-22-33-44-55" 
на соответствующие значения вашего интерфейса и нового MAC-адреса.
